## Instance Variable

- Directly declared inside a class.
- Instance variable memory is allocated on **Heap area** with object/initial.
- Its memory is deallocated by **Garbage Collector** when object become reference less.
- The scope of instance variable is within the class.
- Their accessibility from outside the class is determined by their access modifiers.
- Default value of instance variable will be initialized / given.

---
## Local Variable

- Local variable are directly declared inside method / block / loops within a class.
- Memory is allocated in stack area.
- Memory is deallocated when stack frame of that method / block is cleared after completing its tasks.
- The scope of local variable are limited to method / block / loop where it is declared.
- There is no default value of local variable.
