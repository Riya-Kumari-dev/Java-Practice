# Java

## 📝 Java as a programming Language

- It is created by James Gosling and his team at SunMicrosystems.
- Java is a high level, object-oriented, robust, secure, high performance, multithreaded programming language.
- It is also platform independent that makes it different from other programming language.

## 🚀 Features 

- Object-oriented
- Platform independent
- Robust
- Secure
- Multithreaded 
- Distributed
- Both compiled and interpreted

## Java as "Write Once Run Anywhere"

When Java code is compiled Java compiler converts the Java programs into the class file(Byte Code) which is intermediate language  between the source code and machine code. This byte code can be run on any machine and is not platform specific.

### JVM 

JVM is known as Java Virtual Machine. The JVM provides the runtime environment in which Java bytecode can be executed. JRE is the name of its implementation. For every platform there are different JVMs (so Java is platform independent).

### Difference Between JDK and JVM

The Java Development Kit(JDK) is used for development. JDK contains all the tools, executables and binaries needed to compile, debug, and run a Java programme. JVM provides the runtime environment in which Java bytecode can be executed.

### ❓ Why Java is not a pure object-oriented language ?

Because Java supports primitive data types such as byte, boolean, char, short, int, float, long and double. It is not a pure object-oriented programming language.
